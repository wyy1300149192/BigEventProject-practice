# **大事件项目-练习使用**
## 用到的知识：

1. **Vue 基础**
   + vue-cli、指令、组件、路由、vuex
2. **axios**

    + baseURL、拦截器

3. **element-ui**
   + 安装与配置、常用的组件
4. **npm 与 模块化**
   * 能使用 npm 维护项目中的依赖包
   * ES6 模块化语法

## 学完本项目可以掌握的能力

1. 登录注册的业务实现流程
2. 在项目中使用 vuex 管理全局共享的数据
3. element-ui 组件在实际开发中的应用
4. 文件上传、富文本编辑器在 Vue 项目中的应用
